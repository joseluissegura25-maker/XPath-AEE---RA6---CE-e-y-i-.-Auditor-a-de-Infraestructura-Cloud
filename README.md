# XPath-AEE---RA6---CE-e-y-i-.-Auditor-a-de-Infraestructura-Cloud

¿Cuántas líneas de código habéis ahorrado?
Alrededor de unas 10 o 12 líneas por cada servidor que definamos. Lo mejor es que si el servidor crece, no hay que tocar la estructura principal, solo los grupos. 

¿Qué error os da VS Code si intentáis poner dos servidores con el mismo ID?
El editor marca el segundo ID en rojo y lanza un error de "Identity Constraint Violation". Dice que el valor del ID ya existe en el ámbito del catálogo.
